\doxysubsection{Detailed Description}
A PWM motor object to use with the STM32. 

A motor in this context uses two channels of a PWM-\/enabled timer that is pre-\/ configured at instantiation. The object assumes that the first and last two channels of a timer are to be used as a pair (i.\+e. CH1 \& CH2 or CH3 \& CH4). It contains a the signed duty cycle (fwd/bwd), a reference to the HAL timer, and an indicator of which channels on the timer are used for this motor.
